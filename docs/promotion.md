# Promotion Guide

[English](promotion.md) | [简体中文](promotion.zh-CN.md)

This project is easiest to explain as:

> A local-first Agent Skill that turns one-sentence meal, workout, sleep, and body-state notes into structured health logs, weekly reviews, and visualization-ready data.

## Positioning

Lead with the problem:

- Fitness and meal apps are often too much work.
- People already tell AI agents what they ate or trained.
- This skill turns those messages into local structured records.
- The point is not coaching. The point is low-friction logging and review.

## What To Show

Show the before and after:

```text
Input:
Breakfast was two eggs and a latte. Today I trained back, deadlift 80kg 5x5.

Output:
```

## Launch Post

```text
I built Agent Health Log Skill, a local-first health logging skill for OpenAI Codex, Claude Code, VS Code Agent Mode, and Agent Skills-compatible tools.

Instead of opening a fitness app, you can tell your agent:
"Breakfast was two eggs and a latte. Today I trained back, deadlift 80kg 5x5."

The skill turns that into local structured CSV/Markdown records, then supports weekly reviews, exercise progress checks, and chart-ready data.

It is not a medical or coaching tool. It is for low-friction logging, local data ownership, and periodic review.

GitHub: https://github.com/Jia0029/agent-health-log-skill
```

## What Not To Claim

Avoid claiming:

- Medical advice
- Injury diagnosis
- Rehab plans
- Accurate nutrition database
- Complete app replacement
- Cloud sync

Keep the promise narrow and credible: one-sentence logging, local files, weekly review, chart-ready data.
