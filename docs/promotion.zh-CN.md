# 宣传指南

[English](promotion.md) | [简体中文](promotion.zh-CN.md)

这个项目最容易讲清楚的一句话是：

> 一个本地优先的 Agent Skill，把一句话的饮食、训练、睡眠和身体状态记录，变成结构化健康日志、每周复盘和可视化数据。

## 定位

先讲痛点：

- 传统健身 / 饮食 App 记录成本太高。
- 很多人已经会把今天吃了什么、练了什么告诉 AI Agent。
- 这个 Skill 把这些自然语言变成本地结构化记录。
- 重点不是“健身指导”，而是“低摩擦记录 + 本地数据沉淀 + 周期复盘”。

## 展示什么

展示 before / after：

```text
输入：
早餐两个鸡蛋，一杯拿铁。今天练背，硬拉 80kg 5x5。

输出：
- nutrition.csv
- training.csv
- 每周 Markdown 复盘
- 可用于画图的趋势数据
```

## 中文发布文案

```text
我做了一个 Agent Health Log Skill：给 OpenAI Codex、Claude Code、VS Code Agent Mode 等 AI Agent 用的本地优先健康记录 Skill。

你不用打开健身 App 填表，只要告诉 Agent：
“早餐两个鸡蛋，一杯拿铁。今天练背，硬拉 80kg 5x5。”

它会把自然语言转成本地 CSV / Markdown 结构化记录，后续可以生成周报、查看动作进步，也可以输出可视化图表需要的数据。

它不是医疗工具，也不是健身教练。它解决的是：怎么用最低成本把饮食、训练、睡眠和身体状态长期记录下来，并定期复盘。

GitHub: https://github.com/Jia0029/agent-health-log-skill
```

## 不要这样宣传

不要说它能：

- 提供医疗建议
- 诊断伤病
- 制定康复计划
- 精准计算营养
- 替代完整健身 App
- 自动云同步

宣传时保持承诺清晰：一句话记录、本地文件、每周复盘、可视化数据基础。
